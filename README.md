# 🤖 QR Code Telegram Bot

Бот для быстрого преобразования текста в QR коды с возможностью просмотра и печати.

## ✨ Возможности

- ✅ Преобразует любой текст в QR код
- ✅ Отправляет QR код фото в Telegram
- ✅ Ссылка на веб-интерфейс для просмотра в браузере
- ✅ Возможность печати QR кода
- ✅ Размещение исходного текста под кодом
- ✅ Фильтрация (обрабатывает только текст, игнорирует гифы, стикеры и т.д.)
- ✅ Готов к развертыванию на Railway

## 🛠️ Требования

- Python 3.9+
- Telegram Bot Token (от @BotFather)
- Railway Account (для хостинга)

## 📦 Зависимости

```bash
pip install -r requirements.txt
```

## 🚀 Развертывание на Railway

### 1. Создание бота в Telegram

1. Напишите @BotFather в Telegram
2. Нажмите `/start` и следуйте инструкциям
3. Создайте нового бота (`/newbot`)
4. Скопируйте токен бота

### 2. Развертывание на Railway

1. Перейдите на [railway.app](https://railway.app)
2. Создайте новый проект
3. Подключите ваш GitHub репозиторий
4. В настройках проекта добавьте переменную окружения:
   - `TELEGRAM_TOKEN`: Ваш токен бота (от @BotFather)
   - ⚠️ **ВАЖНО**: Больше ничего не нужно!

5. Нажмите Deploy

### ⚠️ Важно: Polling вместо Webhook

Бот работает в **polling режиме** (безопаснее и надежнее для Railway):
- ✅ Бот сам опрашивает Telegram серверы
- ✅ Не требует webhook конфигурации
- ✅ Работает на стандартных Railway контейнерах
- ❌ Большее использование API (но в рамках free tier)

## 🧪 Локальное тестирование

### 1. Создайте файл `.env`

```bash
cp .env.example .env
```

Отредактируйте `.env` с вашими данными.

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

### 3. Запустите бота

```bash
python main.py
```

**Примечание**: Для локального тестирования нужен publically доступный URL (используйте ngrok или похожий инструмент).

- `GET /qr/<qr_id>` - Просмотр QR кода в браузере
- `GET /qr/<qr_id>/image` - Получить изображение QR кода (PNG)

## 🐛 Решение проблем

### Бот не отвечает на команду /start
- Убедитесь, что `TELEGRAM_TOKEN` установлен в Railway
- Проверьте логи: Railway → Deployments → View Logs
- Ищите строку "✅ Telegram бот запущен в polling режиме!"

### Ошибка: "TELEGRAM_TOKEN не установлен"
- Проверьте, что вы установили переменную окружения `TELEGRAM_TOKEN`
- На Railway это делается в Project Settings → Variables

### Бот не обрабатывает сообщения
- Проверьте логи на Railway (Deployments → View Logs)
- Убедитесь, что нет ошибок при запуске бота
- Убедитесь, что токен скопирован правильно (без пробелов)

### QR код не открывается в браузере
- Если используете Railway: проверьте, что правильно скопирована ссылка
- На локальном хосте: URL должен быть `http://localhost:8443`
- Проверьте консоль браузера (F12) на ошибки

### Как просмотреть логи на Railway?
1. Откройте ваш проект на Railway
2. Перейдите на вкладку **Deployments**
3. Нажмите на последний deployment
4. Нажмите **View Logs** и смотрите ошибки
